# 第 1 章 机器学习导论

## 1.1 什么是机器学习

机器学习（Machine Learning）研究如何从数据中自动学习规律，并利用所学规律对未知数据进行预测或决策。与显式编程不同，机器学习通过**训练**调整模型参数，使模型在任务上表现更好。

## 1.2 学习类型

- **监督学习**：样本带有标签，如分类、回归。
- **无监督学习**：无标签，如聚类、降维。
- **强化学习**：智能体通过与环境交互获得奖励信号进行学习。

## 1.3 基本流程

1. 收集与清洗数据  
2. 特征工程与数据集划分（训练/验证/测试）  
3. 选择模型并训练  
4. 评估与调参  
5. 部署与监控  

## 1.4 关键术语

| 术语 | 含义 |
|------|------|
| 特征 | 描述样本的属性向量 |
| 标签 | 监督学习中的目标值 |
| 过拟合 | 模型在训练集上很好、泛化差 |
| 欠拟合 | 模型过于简单，拟合不足 |

## 1.5 学习目标

学完本章，你应能区分监督/无监督问题，描述标准 ML 流水线，并解释过拟合与欠拟合的含义。
