# 第 2 章 线性回归与梯度下降

## 2.1 线性回归模型

一元线性回归假设目标 \(y\) 与特征 \(x\) 近似满足：

\[
\hat{y} = w x + b
\]

其中 \(w\) 为权重，\(b\) 为偏置。多元情形下，\(\hat{y} = \mathbf{w}^\top \mathbf{x} + b\)。

## 2.2 损失函数

常用均方误差（MSE）：

\[
L(w,b) = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2
\]

训练目标是最小化 \(L\)。

## 2.3 梯度下降

迭代更新：

\[
w \leftarrow w - \eta \frac{\partial L}{\partial w}, \quad
b \leftarrow b - \eta \frac{\partial L}{\partial b}
\]

\(\eta\) 为学习率。批量梯度下降使用全量数据；随机梯度下降（SGD）每次使用小批量，适合大规模数据。

## 2.4 正则化

为缓解过拟合，可在损失中加入 L2 正则：

\[
L' = L + \lambda \|w\|^2
\]

## 2.5 实践要点

- 特征缩放可加速收敛  
- 学习率过大导致震荡，过小收敛慢  
- 使用验证集选择超参数  

## 2.6 学习目标

掌握线性回归假设、MSE 与梯度下降更新式，并能说明正则化的作用。
