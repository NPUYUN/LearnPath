# 机器学习导论 — 课程知识库

本目录为学径系统 RAG 输入，须包含**至少一门完整课程**的讲义/课件文本。

## 目录结构

```
ml_intro/
├── README.md          # 本说明
└── chapters/          # 按章 Markdown（框架已含 2 章样例）
    ├── 01-导论.md
    └── 02-线性回归.md
```

## 采集建议

1. 每章一个 `.md`，文首可用 YAML front matter（可选）。
2. 章节命名：`NN-章节名.md`，便于元数据 `chapter` 字段。
3. 目标规模：≥5–8 章；初赛演示至少完成 2 章并可继续扩充。
4. 运行入库：`python scripts/ingest_kb.py`（从项目根目录执行）。
