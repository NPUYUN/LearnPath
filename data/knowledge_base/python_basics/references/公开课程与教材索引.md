## 推荐练习方向

1. 变量与类型 → 条件与循环小工具（计算器、猜数游戏）
2. 列表/字典 → 成绩统计、词频统计
3. 函数 → 模块化重构重复代码
4. 类 → 模拟图书/学生管理系统
5. 文件与异常 → 日志分析脚本
