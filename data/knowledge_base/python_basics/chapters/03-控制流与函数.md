# 第 3 章 控制流与函数

## 3.1 条件分支

使用 `if` / `elif` / `else` 根据布尔表达式选择执行路径。比较运算符（`==`, `!=`, `<`, `>`）与逻辑运算符（`and`, `or`, `not`）可组合复杂条件。

```python
score = 85
if score >= 90:
    grade = "A"
elif score >= 60:
    grade = "B"
else:
    grade = "C"
```

## 3.2 循环

- **for**：遍历可迭代对象（列表、字符串、range 等）。
- **while**：在条件为真时重复执行，需注意避免死循环。

```python
for i in range(5):
    print(i * i)

n = 3
while n > 0:
    n -= 1
```

`break` 提前结束循环，`continue` 跳过本次迭代。

## 3.3 函数

函数用 `def` 定义，可带位置参数、默认参数与返回值。

```python
def mean(values: list[float]) -> float:
    return sum(values) / len(values)
```

良好实践：函数名动词化、单一职责、文档字符串说明参数与返回值。

## 3.4 作用域

- **局部变量**：在函数内赋值默认可读写的为局部变量。
- **global / nonlocal**：跨层修改外层变量时使用（应尽量少用）。

## 3.5 学习目标

能编写带分支与循环的小程序，将重复逻辑封装为函数，并理解局部作用域。
