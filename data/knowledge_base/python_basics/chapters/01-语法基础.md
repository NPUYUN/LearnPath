# Python 语法基础

## 变量与类型

Python 是动态类型语言。常见内置类型包括 `int`、`float`、`str`、`bool`、`list`、`dict`、`tuple`、`set`。

```python
x = 42
name = "LearnPath"
items = [1, 2, 3]
```

## 控制流

- `if / elif / else` 分支
- `for` 遍历可迭代对象
- `while` 循环需避免死循环

## 函数

使用 `def` 定义函数，支持默认参数与 `*args`、`**kwargs`。

```python
def greet(name: str, prefix: str = "Hello") -> str:
    return f"{prefix}, {name}"
```

## 模块与包

通过 `import` 引入标准库或第三方包；项目内模块使用包路径组织。
