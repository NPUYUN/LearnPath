# 第 4 章 面向对象编程

## 4.1 类与对象

类是创建对象的蓝图；对象是类的实例。

```python
class Student:
    def __init__(self, name: str, sid: str):
        self.name = name
        self.sid = sid

    def greet(self) -> str:
        return f"你好，我是 {self.name}"

s = Student("小明", "2024001")
print(s.greet())
```

## 4.2 封装

用单下划线 `_attr` 表示“内部使用”，双下划线可触发名称改写（name mangling）。通过 `@property` 暴露只读或受控访问。

## 4.3 继承

子类复用父类行为并可重写方法。

```python
class Animal:
    def speak(self) -> str:
        return "..."

class Dog(Animal):
    def speak(self) -> str:
        return "汪汪"
```

多继承需谨慎，优先组合优于复杂继承树。

## 4.4 特殊方法

`__str__`、`__repr__`、`__len__` 等让对象与内置函数协同工作，提升可读性。

## 4.5 学习目标

能设计简单类层次，区分组合与继承，并编写符合 Python 风格的 OOP 代码。
