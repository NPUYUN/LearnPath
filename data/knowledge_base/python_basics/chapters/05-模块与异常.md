# 第 5 章 模块、包与异常处理

## 5.1 模块导入

```python
import math
from pathlib import Path

print(math.sqrt(2))
```

模块搜索路径由 `sys.path` 决定；项目内包用目录 + `__init__.py`（Python 3.3+ 可用 namespace 包）。

## 5.2 标准库速览

| 模块 | 用途 |
|------|------|
| `os` / `pathlib` | 路径与文件系统 |
| `json` | JSON 读写 |
| `datetime` | 日期时间 |
| `collections` | 高级容器 |
| `itertools` | 迭代工具 |

## 5.3 异常处理

```python
try:
    x = int(input("数字: "))
except ValueError:
    print("请输入有效整数")
else:
    print("平方:", x * x)
finally:
    print("结束")
```

只捕获预期异常类型，避免裸 `except:` 吞掉所有错误。

## 5.4 文件 I/O 简述

```python
from pathlib import Path

p = Path("notes.txt")
p.write_text("hello\n", encoding="utf-8")
text = p.read_text(encoding="utf-8")
```

大文件推荐逐行读取或使用上下文管理器。
