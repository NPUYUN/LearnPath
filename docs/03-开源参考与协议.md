## 4.  Attribution 模板（答辩 PPT / 说明书用）

> 本系统基于 LangGraph 构建多智能体协同框架，使用 ChromaDB 实现课程知识检索，大语言模型能力由科大讯飞星火认知大模型提供（开发阶段可选用 Mock 模式）。前端采用 Next.js。参考了 gaokao_tutor、edu-agent 等开源项目的架构思路，核心代码为团队自主开发。
